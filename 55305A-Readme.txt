*****   Class Delivery   ***** 
55305A is designed to help those new to databases to learn the fundamentals of modelling and creating new databases.
It has a modular design allowing the lab exercises to be taught in any order the trainer chooses.  This also allows customized deliveries of the material in combination with other Azure, PowerShell or database classes.  
For additional help in configuring standard or customized deliveries, please read the Trainer Preparation Guide (55305A-ENU_TrainerPreparationandNotes.pdf) or contact the author at (neil@neiltucker.biz).
